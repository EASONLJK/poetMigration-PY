# 本地到github
# 1、git init 初始化本地项目

# 2、git remote add origin+你远程仓库的地址 链接远程仓库

# 3、输入git add . 提交整个项目

# 4、输入git commit -m ‘update’ 更新项目

# 5、输入git push -u oringin main 提交到github main分支

# 6、打开GitHub查看是否成功

# 如果git push 不成功 使用git push -f origin main
